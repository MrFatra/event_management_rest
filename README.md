# Event Management REST

## Tentang Proyek

Project ini dirancang untuk menjadi pusat informasi dan pendaftaran berbagai kegiatan di lingkungan kampus menggunakan REST API. Aplikasi ini bertujuan untuk memudahkan mahasiswa, dosen, dan staf dalam menemukan serta berpartisipasi dalam acara-acara pengembangan diri seperti seminar, webinar, dan workshop.

## Fitur Utama

- **List Katalog Acara**

- **Manajemen Pengguna & Otentikasi**

- **Pendaftaran Acara**

- **Sistem Ulasan & Peringkat**

## Planning

- **Integrasi Payment Gateway:** Untuk kedepannya saya ingin menambahkan payment gateway ke dalam sistem ini, dengan begitu pengguna dapat dengan mudah bertransaksi pada aplikasi nya secara langsung untuk acara berbayar.
